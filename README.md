#Android App快速开发框架
基于此框架可以快速开发出一款APP，不重复造轮子。

## 特性
- 基于MVVM + OKHttp + Fresco(Glide)
- 提供列表页面的Base组件
- 提供分页列表页面的Base组件（内置了下拉刷新和上拉加载更多）
- 提供Tab页面的Base组件
